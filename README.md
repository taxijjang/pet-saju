# 댕냥

강아지와 고양이의 생일, 태어난 시간, 성격 키워드를 바탕으로 귀엽고 가볍게 "재미로 보는 사주 카드"를 보여주는 정적 웹사이트입니다.

## 실행 방법

별도 빌드 없이 `index.html`을 브라우저에서 바로 열어도 됩니다.

로컬 서버로 확인하고 싶다면 아래처럼 실행할 수 있습니다.

```powershell
python -m http.server 8000
```

그 다음 브라우저에서 `http://localhost:8000`으로 접속하면 됩니다.

## 구성

- `index.html`: 한 페이지 UI 구조
- `styles.css`: 귀엽고 따뜻한 톤의 스타일과 반응형 레이아웃
- `script.js`: 입력값 기반 사주 카드 생성 로직

## 참고

이 프로젝트의 결과는 정통 역술 해석이 아니라 재미를 위한 가벼운 콘텐츠입니다.
